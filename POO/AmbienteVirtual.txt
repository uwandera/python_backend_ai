Para gerenciar um ambiente virtual para uso de pacotes antigos e versionamento de código obsole

usar:
python 3 -m venv <nome do projeto>

source <nome do projeto>/bin/activate
comando não funciona nesta versão de python,
tive que fazer uma mudança de permissão para rodar o comando:
    activate <nome do ambiente virtual>

SQL:
    Structured Query Language:
        DQL: Data Query Language:
            SELECT >>> consulta de dados, recuperar informação armazenada no banco

        DML: Data Manipulation Language:
            INSERT, UPDATE, DELETE >>> consultar, alterar e exluir informação

        DDL: Data DEfinition Language:
            CREATE, ALTER, DROP >>> estruturação da tabela, onde sera armazenada, quais tipos de dados
        
        DCL: Data Control Language:
            GRANT, REVOKE >>> permissionamentos de usuarios para manipulação de dados

        DTL: Data Transaction language:
            BEGIN, COMMIT, ROLLBACK >>> garantia da execução das propriedades anteriores
